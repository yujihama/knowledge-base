## 要約

フローベースのVision-Language-Action（VLA）ポリシーは、ロボット制御においてアクション生成の高い表現力を持つが、ガウスノイズから有意なアクション構造を復元するために多ステップの推論（NFE: Number of Function Evaluations）が必要となり、リアルタイム制約下での効率と品質のトレードオフが課題となっている。本論文ではこの問題に対し、生成の出発点そのものを再設計するアプローチ「CF-VLA（Coarse-to-Fine VLA）」を提案する。

CF-VLAは2段階の生成フレームワークを採用する。第1段階（粗段階）では、エンドポイント速度の条件付き事後分布を学習し、ガウスノイズを「アクション認識済み初期点」へと変換する。第2段階（細段階）では、この構造化された初期点から固定時刻の局所的細修正を1ステップで実施し残差誤差を補正する。サンプリング軌跡を単に短くするのではなく、情報を持たない出発点を情報豊富な出発点に置き換えることで、少ないNFEでも高品質なアクションを生成できる。

訓練安定化のため、まず制御された粗予測器を学習し、次に両段階を共同最適化するステップワイズ戦略を導入している。実験はロボット操作ベンチマークのCALVINとLIBEROで実施。低NFEレジーム（NFE=2）における既存手法をすべて上回り、NFE=10のπ₀.₅ベースラインと同等以上の性能を複数指標で達成。アクションサンプリングのレイテンシを75.4%削減し、実機ロボットでの平均成功率83.0%を記録（MIPより+19.5ポイント、π₀.₅より+4.0ポイント）。

この結果は、生成過程の出発点を構造化することがVLAポリシーの推論効率と性能を同時に高める上で有効であることを示している。コードはGitHubで公開済み（EmbodiedAI-RoboTron/CF-VLA）。監査エージェント開発への直接的示唆は薄いが、LLMベースのエージェントにおける推論ステップ削減・効率化（例：ActionやChain-of-Thoughtの粗→細設計）への応用アイデアが得られる。

## アイデア

- ガウスノイズという「無構造な出発点」を「アクション認識済み出発点」に置き換えるだけで、サンプリングステップを大幅削減できるという逆転の発想は、LLMのデコーディング効率化にも応用可能
- 粗予測器を先に学習してから共同最適化するステップワイズ訓練戦略は、複雑な多段階モデルの訓練安定化手法として汎用的に参考になる
- NFE=2で実機成功率83.0%という結果は、リアルタイムロボティクスにおける「推論速度と精度のフロンティア」を塗り替えており、エッジデバイス上のAIエージェント展開に直結する知見
