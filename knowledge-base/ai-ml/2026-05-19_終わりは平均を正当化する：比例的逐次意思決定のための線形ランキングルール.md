## 要約

本論文は、AIアライメントと参加型設計を動機として、「繰り返し使用する意思決定ルールをどのように集合的に選択するか」という民主的設計問題を研究する。具体的には、バッチ X=(x_1,...,x_m) 内のアイテムを固定スコアリングベクトル θ* によるスコア ⟨θ*,x_j⟩ でランク付けする「線形ランキングルール」を対象とする。

各有権者 i が自分の好むスコアリングベクトル θ^(i) と人口比率 α^(i) を持つとき、集合的なベクトル θ* に求める性質として「個別比例性（IP: Individual Proportionality）」を定義する。これは、各有権者タイプ i が結果のランキングに対して α^(i) に比例した程度で同意できることを指す。この比例性には2種類あり、長期平均での一致を求める「Long-run IP」と、各バッチ内での一致を求める「Per-batch IP」がある。

従来のデフォルトルールである「算術平均（θ^(i) の単純平均）」は、多数派優位（majoritarian）であることが既知であり、多様な有権者の意見を公平にバランスさせられるか自体が不明だった。本研究の主要結果は、「角度平均（angular mean）」——算術平均の球面版、つまり各 θ^(i) を単位球面上に正規化してから平均し、再び正規化したもの——が Long-run IP を満たすという驚くべき事実の証明である。

一方、固定線形ルールによる厳密な Per-batch IP は不可能であることも示した。ただし、Per-batch IP と Long-run IP のギャップはバッチサイズの増加とともに急速に縮小する。3つの実世界の選好データセットを用いた実験では、有権者の選好が均質なケースでは全ルールが同様の性能を示す一方、高い意見不一致レジームでは角度平均が比例性を大幅に改善することが確認された。

監査エージェント開発への示唆：複数のステークホルダー（部門、監査人、経営陣）が異なる優先基準を持つ状況で、共通のスコアリングルールやランキング方式を選定する際に、この「角度平均」アプローチが公平性担保の手法として応用できる可能性がある。特に、長期的な監査方針決定や優先順位付けルールの設計において、多数派バイアスを排除しながら各ステークホルダーの意見を比例的に反映するメカニズムとして参考になる。

## アイデア

- 算術平均という直感的なデフォルトが多数派優位であるという欠陥を持つのに対し、単純な正規化操作（角度平均）だけで Long-run 比例性を達成できるという理論的な意外性
- Per-batch IP が固定線形ルールでは原理的に不可能であるにもかかわらず、バッチサイズを大きくするだけでそのギャップを実用的に無視できるレベルまで縮小できるという実用的な脱出路の存在
- 参加型AI設計（participatory AI）において、ルール選択そのものを民主的プロセスとして定式化するフレームワークは、LLMの出力評価基準やRAGの検索ランキング設計における多様な利害関係者調整に転用できる可能性
