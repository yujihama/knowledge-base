## 要約

本論文は、深層Kolmogorov-Arnold Network（KAN）の理論的基盤を強化する研究で、組成的に構造化された連続関数のKAN表現における「層別リプシッツ積（Layerwise Lipschitz Product）」の制御問題を解決する。

KANはMLPの代替として注目されるアーキテクチャで、各エッジに学習可能な一変数関数を配置する。しかし深いKANスタックでは、各層のリプシッツ定数の積が指数的に爆発するリスクがあり、Liu et al.（2024）がこのギャップを指摘していた。本研究はその問題に正面から取り組む。

主な理論的成果は以下の通り。まず、[0,1]^nからRへの連続関数fがN個の内部ノードと組成スパース性s=O(1)を持つ有限計算木で表現可能であれば、深いKAN表現が存在することを証明した。各内部ノードはブロック深さとリプシッツ積が制御されたプリミティブKANブロックで実現される。

層別リプシッツ積に関して、入力次元nに依存しないドメイン依存境界P(KAN_f) <= max(C*,1)^L_f（L_f <= c_max * N）を導出した。特に重要な結果として、{+, -, ×, sin, cos}の標準演算のみを用い、[0,1]有界入力を持つ場合、P(KAN) <= 1が成立する。これはリプシッツ積が爆発しないことを意味し、深いKANスタックが安定的に動作することの理論的保証となる。

層幅についてはn_l <= n + 2*w_max*Nという境界が示され、一様近似誤差はN * max(C*,1)^d(f) * epsilon_Opで抑えられる。f ∈ C^mの場合はBスプラインの最適近似レートも達成される。加算木に対するレンジ境界（B_f <= N+1）も導出されている。

実験では組成構造を持つ複数の関数に対してP(KAN)=1.0が確認されており、理論と実験の一致が示されている。この結果は、KANが深くなるほど不安定化するという懸念を払拭し、MLPの代替として実用的な深層KAN設計の指針を提供する。監査エージェント開発への直接的示唆としては、監査ロジックが演算の組成として記述できる場合に、KANベースのモデルが解釈可能性と安定性を両立した選択肢になり得る点が挙げられる。

## アイデア

- 標準的な算術演算とsin/cosのみで構成される計算木に対してリプシッツ積が1以下に保たれるという結果は、KANの安定性を保証する実用的な設計指針となる
- 入力次元nに依存しない層別リプシッツ境界の導出により、高次元関数の近似でもKANが爆発しないという次元の呪いへの対抗軸が生まれる
- 有限計算木という表現クラスと組成スパース性s=O(1)の組み合わせが、複雑な関数を構造的に分解してKANで表現する際の理論的フレームワークとして機能する
