## 要約

Technology Innovation Institute（TII）が2024年5月に公開したFalcon 2は、11Bパラメータの基盤LLMとビジョン言語モデル（VLM）の2モデルで構成される。LLMは5500億トークン（RefinedWeb英語・欧州10言語、技術データ、コード、会話データ）で4段階のカリキュラム学習により訓練された。コンテキスト長は段階的に2048→4096→8192トークンへ拡張され、最終ステージでは高品質データのみを使用して性能強化を図った。アーキテクチャはTransformerブロック60層、クエリヘッド32・KVヘッド8のGrouped Query Attention（GQA）、ヘッド次元128、並列アテンションを採用。1024台のA100 40GB GPUで3Dパラレリズム（TP=8、PP=1、DP=128）とZeRO、Flash-Attention 2を組み合わせてbfloat16精度で訓練した。Open LLM Leaderboard評価では平均64.28点を記録し、3倍のデータで訓練されたLlama3-8B（62.38点）やMistral-7B（60.97点）を上回り、Gemma-7B（64.29点）と同等。特筆すべきはゼロショット評価でFalcon-40Bと同等性能を4分の1のモデルサイズで達成している点。多言語性能はドイツ語・スペイン語・フランス語・イタリア語・オランダ語・ルーマニア語でFalcon-40Bを全言語で上回る。コード生成（HumanEval Python pass@1）は29.59%。VLMはCLIP ViT-L/14ビジョンエンコーダをFalcon2-11B chatファインチューン済みモデルと統合し、高解像度での動的エンコードメカニズムにより小物体の細粒度認識を強化している。ライセンスはTII Falcon License 2.0（ほぼ商用利用可能）。

## アイデア

- 4段階カリキュラム学習でコンテキスト長を段階拡張しつつ最終ステージに高品質データを集中投入することで、トークン数を抑えながら高性能を達成する訓練戦略
- Grouped Query Attention（KVヘッド数をクエリの1/4に削減）により推論時のKVキャッシュメモリを大幅削減し、11Bながら低コスト推論を実現している点
- 11B LLMをベースにCLIPビジョンエンコーダを統合したVLMアーキテクチャ：テキスト基盤モデルをマルチモーダルに拡張する際の段階的エコシステム構築アプローチ
