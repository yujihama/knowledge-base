## 要約

本論文は、小売ECにおける「次バスケット再購入推薦」の課題に取り組む新アーキテクチャCASE（Cadence-Aware Set Encoding）を提案する。再購入行動（同一商品を定期的に買い直すパターン）は小売推薦の主要シグナルであるにもかかわらず、既存モデルの多くは購入履歴をバスケット訪問順の離散イベント列として表現するため、カレンダー時間の経過や商品ごとの購入リズム（ケイデンス）を明示的にモデル化できていない。

CASEの技術的な核心は3点ある。①各商品の購入履歴を固定ホライゾン（例：過去90日）のカレンダー時間シグナルとして表現することで、「最後の購入から何日経過したか」を直接エンコードする。②マルチスケール時間畳み込み（Multi-Scale Temporal Convolution）を共有パラメータで適用し、週次・月次・季節性など異なる粒度の購入リズムを同時に捉える。③Induced Set Attention（誘導セットアテンション）により商品間の依存関係をサブ二乗複雑度（O(n·m)、mはインデュースドポイント数）で学習し、数千万ユーザー・大規模商品カタログでもバッチ推論を実用的なコストで実現する。

ケイデンス学習と商品間インタラクションを分離（decouple）した設計により、カレンダー時間モデリングとスケーラビリティを同時に達成している点が新規性の中心。

評価結果は3つの公開ベンチマーク（明示されていないが食品・日用品系データセットと推測）と自社独自データセット（Walmart規模と思われる）で実施。数千万ユーザー・大規模商品カタログの本番環境では、Top-5でPrecision +8.6%・Recall +9.9%の相対改善を達成した。NDCG等複数指標でも強力なベースライン（TIFU-KNN、DNNTSP等）を一貫して上回る。

## アイデア

- カレンダー時間を連続シグナルとして直接エンコードする発想は、監査ログの時系列分析（取引の定期性・異常頻度検知）にも転用できる
- Induced Set Attentionによるサブ二乗複雑度の実現は、大規模エージェントシステムでの多エンティティ間依存関係モデリングのスケーラビリティ問題への一解法になりうる
- 「ケイデンス（周期性）学習」と「クロスアイテム依存関係」を意図的に分離した設計思想は、モジュール性の高いLangGraphエージェント設計と共鳴する概念的アプローチ
