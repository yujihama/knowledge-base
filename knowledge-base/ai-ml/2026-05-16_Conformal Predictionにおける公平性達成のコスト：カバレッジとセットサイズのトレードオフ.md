## 要約

Conformal Prediction（CP）は、機械学習モデルの予測に統計的な不確実性推定を付与する手法であり、事前に設定したカバレッジ率（例：90%）で真の答えを含む予測セットを返すことを保証する。通常の実装では、キャリブレーション時に全グループのスコアを混合した単一の閾値（プール閾値）を使用するが、本論文はこのアプローチが集団間の不均質性を隠蔽し、グループごとのカバレッジに歪みをもたらすことを理論的に示している。

主要な貢献は3点。第一に、プール校正が生む歪みに関する「保存則」と下限を導出した。この下限は、グループ間の分位点の異質性（cross-group quantile heterogeneity）によって決まる不可約な歪みが存在することを示す。つまり、どれだけ工夫しても、グループ間のスコア分布が異なる限り、何らかの歪みは残り続ける。

第二に、CPにおける公平性の代表的な2定義—「Equalized Coverage（グループ間でカバレッジを均等化）」と「Equalized Set Size（グループ間で予測セットサイズを均等化）」—が根本的に両立しないことを証明した。一方を最適化すると、他方が劣化するという構造的なトレードオフが存在する。

第三に、グループを分離して校正するポリシー（グループ別閾値）と混合するポリシー（プール閾値）の間を移行するコストを定量化した。合成データおよび実データを用いた実験で、有限サンプル校正後も双方向のトレードオフが確認された。

結論として、校正方針の選択はグループ間の異質性を除去するのではなく、その歪みをカバレッジ次元とセットサイズ次元のどちらに現れさせるかを決定するに過ぎない。これは、公平性を指向した校正設計における原理的な分析フレームワークを提供するものである。監査AIへの示唆としては、グループ属性（業種、規模、リスクカテゴリ等）に基づいてモデル予測の不確実性推定を設計する際、単一の閾値を用いると特定グループで過剰または過小なカバレッジが生じるリスクがあり、グループ別校正の採用とそのコストの明示的な評価が重要になる。

## アイデア

- 「保存則」の発見：プール校正における歪みは除去不可能であり、校正方針はその歪みの現れる次元（カバレッジ vs セットサイズ）を選択しているに過ぎないという洞察
- 2つの公平性定義（Equalized Coverage / Equalized Set Size）が数学的に両立しないことの証明は、AI倫理・公平性研究における「公平性の不可能定理」の系譜に位置づけられる
- グループ別校正とプール校正のコスト定量化フレームワークは、実務でどの公平性基準を優先すべきかの意思決定を支援するツールとして応用可能
