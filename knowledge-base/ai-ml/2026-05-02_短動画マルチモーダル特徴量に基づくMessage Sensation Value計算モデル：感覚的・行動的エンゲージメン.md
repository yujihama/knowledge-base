## 要約

本研究は、短動画における視聴者エンゲージメントを予測する計算モデルを、Message Sensation Value（MSV）理論に基づいて構築・検証したものである。MSVとはメッセージの感覚的刺激強度を定量化する概念で、映像・音声・テキストなどマルチモーダルな特徴量から算出される。

研究では1,200本の短動画を対象に、マルチモーダル特徴量の自動抽出と人手評価を組み合わせてMSVモデルを構築した。モデルの汎化性能は3つの短動画プラットフォームから収集した合計14,492件の未知データセットで検証された。

主要な発見として、MSVと感覚的エンゲージメント（視聴完了率・注意保持など）の間には正の線形関係が認められた一方、行動的エンゲージメント（いいね・シェア・コメント等）との関係は逆U字型を示した。すなわち、MSVが高いほど強い感覚刺激をもたらすが、行動的エンゲージメントを最大化するのは中程度のMSVであるという知見が得られた。過剰な刺激は視聴者の行動を抑制する可能性を示唆している。

技術的には、映像の動き量・色彩変化・音圧・字幕の感情強度などマルチモーダルな特徴量を統合してMSVスコアを算出し、それをエンゲージメント指標の予測変数とする回帰モデルを構成している。大規模な外部データセットでの検証によりモデルの堅牢性が確認されており、プラットフォーム横断での一般化可能性も示された。

監査AI・内部統制との直接的な関連は薄いが、マルチモーダルデータからユーザー行動を予測するモデル設計の方法論は、監査エージェントがレポートや説明資料の「伝達効果」を定量評価する際の参考になりうる。また、LLM-as-judgeフレームワークにおける人手評価とモデル評価の組み合わせ手法は、監査エージェントの評価設計にも応用可能な観点を含む。

## アイデア

- MSVと行動エンゲージメントの逆U字関係：刺激が強すぎると行動（いいね・シェア）が抑制されるという非線形性は、コンテンツ最適化モデル設計において単純な「刺激最大化」が逆効果になることを示す
- 1,200本の人手評価＋14,492件の自動検証という二段階バリデーション設計：小規模人手ラベルでモデルを構築し大規模自動データで汎化性を確認する手法は、ラベルコストの高い専門ドメイン（監査・医療等）でのモデル開発にも適用可能
- マルチモーダル特徴量の統合によるユーザー反応予測：映像・音声・テキストを横断した特徴量エンジニアリングで人間の主観的反応を定量化する枠組みは、説明文書やダッシュボードの「伝わりやすさ」評価への転用が考えられる
