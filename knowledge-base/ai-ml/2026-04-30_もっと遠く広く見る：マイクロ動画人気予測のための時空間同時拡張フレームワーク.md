## 要約

マイクロ動画人気予測（MVPP: Micro-Video Popularity Prediction）は、TikTokやReelsのような短尺動画プラットフォームにおいて、動画の将来的な再生数・エンゲージメントを事前に予測するタスクであり、コンテンツ推薦やトラフィック配分の最適化に不可欠な技術である。本論文では、既存アプローチが抱える2つの根本的な制約を同時に解決する統合フレームワーク「Joint Spatio-Temporal Enlargement」を提案する。

【時間軸の問題と解決策】既存手法は動画フレームのスパースサンプリング（例：等間隔に数フレームのみ抽出）に依存しており、ハイライトシーンや急激な展開変化を見逃すという短距離知覚の限界があった。本手法ではTemporal Enlargementとして、フレームスコアリングモジュールを導入。（1）スパースサンプリングによる全体的な内容把握と、（2）デンスパーセプション（密な知覚）によるローカルな詳細抽出という2つの補完的パスウェイを用意し、その出力をアダプティブに融合することで、極めて長いビデオシーケンスの堅牢な理解を実現する。

【空間軸の問題と解決策】既存手法は過去動画との関連性を「フラットな検索メモリ」で管理しており、容量制限と検索効率の低さが拡張性のボトルネックとなっていた。本手法ではSpatial Enlargementとして、Topology-Aware Memory Bank（トポロジー認識型メモリバンク）を構築。過去の関連動画をトポロジカルな関係性に基づいて階層的にクラスタリングし、新規動画追加時には動画そのものではなく対応クラスターのエンコーダー特徴量を更新する。これにより、ストレージ容量を無制限に増やすことなく、無制限の履歴関連付けを実現する。

実験では3つのMVPPベンチマークデータセットを用い、11の強力なベースラインと比較。予測精度と順位整合性（ランキング一貫性）の両指標で一貫した性能向上を達成した。監査AI開発への示唆として、Topology-Aware Memory Bankの設計思想——新規データ追加時にエンティティ本体ではなくクラスター表現を更新することで無制限拡張を実現する手法——は、監査エージェントが蓄積する事例ベースの知識管理（継続学習下でのRAG拡張）に応用可能な設計パターンである。

## アイデア

- 新規データ追加時にデータ本体ではなくクラスター表現のみを更新するという設計により、ストレージ増加なしに無制限の履歴統合を実現する点——これはRAGのメモリ管理における一般解として応用価値が高い
- スパースサンプリング（全体把握）とデンスパーセプション（局所詳細）の2経路をアダプティブに融合する設計は、長文書処理における粗粒度・細粒度の特徴融合戦略に直接対応する
- トポロジカル関係性に基づく階層的クラスタリングにより検索空間を構造化する発想は、フラットなベクトルDBの検索効率問題を解決するグラフベースインデックス（HNSWなど）の思想と共鳴する
