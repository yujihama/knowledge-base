## 要約

本記事は、数理最適化や機械学習の理論的基礎となるベクトル微分（スカラーのベクトルによる微分、ベクトルのベクトルによる微分）を、Jacobi行列と勾配を明確に区別する流儀で解説したものである。

【Jacobi行列と勾配の区別】スカラー値関数 f: R^n → R に対して、各偏微分を「行ベクトル」として並べたものをJacobi行列（∂f/∂x）、「列ベクトル」として並べたものを勾配（∇f）と定義し、両者は転置の関係にある（∇f = (∂f/∂x)^⊤）。多くの教材ではこの2つを同一視するが、本記事ではベクトル値関数への拡張時の混乱を防ぐため厳密に区別している。

【ベクトル値関数のJacobi行列】f: R^n → R^m の場合、Jacobi行列は m×n 行列となり、各成分 f_i の偏微分を行として積み上げた構造を持つ。スカラー値の場合はこの特殊ケース（m=1）である。

【連鎖律】Jacobi行列の積として連鎖律 ∂z/∂x = (∂z/∂y)(∂y/∂x) が自然に導かれ、これはニューラルネットワークの誤差逆伝播の数学的根拠となる。

【Hesse行列】勾配 ∇f のJacobi行列として定義される n×n 対称行列 H_f は、関数の凸性判定に用いられる。∇²f という表記は誤解を招くとして、著者は ∂(∇f)/∂x の表記を推奨している。

【例題】(1) 線形関数 f(x) = a⊤x の勾配 ∇f = a、Hesse行列 H_f = O。(2) アフィン写像 f(x) = Ax のJacobi行列 = A。(3) 二次形式 f(x) = x⊤Ax の勾配 ∇f = (A + A⊤)x、Hesse行列 H_f = A + A⊤（A対称の場合、∇f = 2Ax, H_f = 2A）。(4) Sigmoid合成関数 f(x) = σ(a⊤x) の勾配 ∇f = f(x)(1-f(x))a（連鎖律を適用）。

参考文献として Mathematics for Machine Learning（Deisenroth et al., Cambridge, 2020）が挙げられており、学部1年生レベルの線形代数・微積分知識を前提とする入門的内容である。

## アイデア

- Jacobi行列（行ベクトル）と勾配（列ベクトル）を転置関係として明確に区別することで、ベクトル値関数への拡張時に次元の混乱が生じないという設計思想は、コードの型安全性と同じ哲学であり実装時のバグ防止に直結する
- 連鎖律をJacobi行列の行列積として表現することで、ニューラルネットワークの誤差逆伝播が単なる行列乗算の連鎖であることが明確になり、自動微分ライブラリ（PyTorch等）の内部動作の直感的理解につながる
- Hesse行列を∇²fでなく∂(∇f)/∂xと書くべきという主張は、曖昧な記法が実装ミスを招く実例であり、数学的厳密性がシステム設計の信頼性に与える影響を示す
