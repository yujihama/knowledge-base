## 要約

自律AIエージェントはプロンプトインジェクション・メモリポイズニング・サプライチェーン攻撃・ソーシャルエンジニアリングといった脅威にさらされているが、既存の防御はプラットフォーム境界に限定されており、エージェント自身の脅威判断能力は未訓練のままである。ClawdGoはこの問題を「内生的セキュリティ認識訓練（Endogenous Security Awareness Training）」として定式化し、モデル重みの変更なしに、推論時点でエージェント自身が脅威を認識・推論できるよう訓練するフレームワークである。

4つの主要コントリビューションがある。①TLDT（Three-Layer Domain Taxonomy）は「自己防衛（Self-Defence）」「オーナー保護（Owner-Protection）」「エンタープライズセキュリティ（Enterprise-Security）」の3層にわたる12の訓練次元を整理した分類体系。②ASAT（Autonomous Security Awareness Training）は、エージェントが攻撃者・防御者・評価者の役割を交互に担うセルフプレイループで、弱い次元から優先的に訓練するweakest-firstカリキュラムスケジューリングを採用。③CSMA（Cross-Session Memory Accumulation）は4層の永続メモリアーキテクチャとAxiom Crystallisation Promotion（ACP）によってセッションをまたいでスキル向上を蓄積する仕組み。④SACP（Security Awareness Calibration Problem）は内生的訓練が引き起こす精度・再現率のトレードオフを形式化した概念である。

実験結果では、weakest-first ASATが平均TLDTスコアを80.9から96.9へと16セッションで向上させ、ランダムスケジューリングに比べて6.5ポイント上回り、12次元中11次元をカバーした。CSMAはセッション間で訓練成果を完全に保持する一方、コールドスタートでの復元は2.4ポイントにとどまり13.6ポイントの差が生じた。一方、SACP現象として、過剰訓練されたエージェントが160件中30件で正当な能力評価をプロンプトインジェクションと誤分類する過検知が観察された。監査エージェント開発への示唆として、SACPが示す精度・再現率のトレードオフは監査判断の誤検知コストと見落としコストのバランスと構造的に同質であり、内生的訓練の強度設定に注意が必要である。また、CSMAの永続メモリ設計はLangGraphでの長期エージェント状態管理に応用できる可能性がある。

## アイデア

- エージェントが攻撃者・防御者・評価者の3役を自己再生するASATループは、外部データなしにセキュリティ判断を向上させる新しい訓練パラダイムであり、強化学習のセルフプレイ手法のセキュリティ応用として興味深い
- SACP（Security Awareness Calibration Problem）は訓練強度が上がるほど正常な操作を攻撃と誤認する過検知が増加するという精度・再現率トレードオフを形式化しており、セキュリティ訓練の限界を理論的に定義した点が重要
- CSMAのAxiom Crystallisation Promotion（ACP）によるセッションをまたいだスキル蓄積は、エージェントの長期記憶設計として汎用的に応用可能であり、コールドスタートとの13.6ポイント差がその有効性を定量的に示している
