## 要約

本稿はThe Gradientに掲載された長編論考（約35分読）で、AIアライメントの根本的な枠組みを問い直す。中心的主張は「合理的な人間は目標（goals）を持たない」というものであり、AIも同様に目標ベースの合理性ではなく「実践（practice）ベースの合理性」を持つべきだと論じる。

従来のAIアライメント議論は「直交性テーゼ（Orthogonality Thesis）」——知能とゴールは独立変数であり、どんな知能レベルでも任意のゴールを持ちうる——を前提に、いかにして正しいゴールをAIに与えるかに注力してきた。しかし著者はこの枠組み自体がEffective Altruism的な効用最大化（consequentialist optimization）を人間の合理性の雛形として暗黙に採用しており、これがアライメント問題の根深い「型不一致（type mismatch）」を生んでいると指摘する。

著者が代替として提案するのが「eudaimonic rationality（幸福論的合理性）」である。これはアリストテレスのeudaimonia（人間の能動的・合理的繁栄）概念に基づく。この枠組みでは、合理的行為は「目的に向かう手段」ではなく「実践のネットワークの構成要素」として理解される——音楽における一音符、計算における一ステップ、細胞活動における一瞬間と同様に。実践（practice）は「高スコアな行為が将来の高スコアな行為を因果的に促進する」ネットワークであり、「数学的卓越を数学的に推進する」「親切さを親切に推進する」という「xをx的に促進する」公式で特徴づけられる。

具体例としてテリー・タオの数学研究が挙げられる。数学者が目指すのは「数学的卓越さ」という固定ゴールではなく、卓越した数学的行為を通じて数学的実践そのものを深化させることだ。この構造は「terminal value（終端価値）」と「instrumental value（手段的価値）」の厳密な区別をほぼ無意味にする。

AIアライメントへの含意として、harmlessnessやcorrigibilityといった安全性概念は、ゴールや規則として実装すると脆弱・不安定になるが、実践ネットワークのダイナミクスとして実装すれば自然で安定的になると主張する。また、eudaimonic agentは学習可能性・安定性・アルゴリズム複雑性の低さという点で「自然な（natural）」エージェント型である可能性があり、MLの訓練プロセスのターゲットとして適していると論じる。監査エージェント開発への示唆として、規則ベースまたはゴールベースの設計よりも、「監査実践のネットワーク」として監査行為を実装する設計思想が、より安定的で倫理的なエージェントを生む可能性がある。

## アイデア

- 「xをx的に促進する（promote x x-ingly）」という公式がエージェント設計の原則として機能しうる点：helpfulnessを帮助的に促進し、honesty を正直に促進するエージェントは、単一ゴール最適化エージェントよりも値整合的に振る舞う可能性がある
- terminal/instrumental value の区別がeudaimonic agentには「不自然」という主張は、LLMの訓練でRLHFやGRPOが報酬シグナルとして何を最適化すべきかという問いに直接接続する——固定スカラー報酬ではなく実践スコアの構造を学習させるべきか
- 直交性テーゼへの反論として、「自然なエージェント型（natural agent type）」という概念を導入し、ML訓練の収束先として eudaimonic rationality が consequentialist rationality と同等以上に現れやすいと主張する点——これはモデルのinductive biasに関する経験的仮説でもある
