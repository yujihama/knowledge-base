## 要約

本論文はThe Gradientに掲載された哲学的AIアライメント論文であり、「合理的な人間はゴールを持たない」という主張から出発する。著者は、人間の合理的行動は最終目標への最適化ではなく、「プラクティス（practices）」——行動・行動傾向・行動評価基準・行動リソースからなるネットワーク——への整合によって成立すると論じる。このプラクティスベースの枠組みを「ユーダイモニック合理性（eudaimonic rationality）」と呼び、アリストテレスの「エウダイモニア（eudaimonia：活発で合理的な人間の繁栄）」に由来する概念として定式化する。

論文の核心的批判は、Effective Altruism（EA）型の功利最大化モデルがAIアライメントの標準パラダイムとして機能していることへの異議申し立てである。EA型最適化エージェントは「手段／目的」の明確な分離を前提とし、あらゆる価値を最終的な効用関数に還元しようとする。著者はこれが人間の価値構造との「型の不一致（type mismatch）」を生み出し、harmlessness・corrigibility・helpfulnessといった安全特性でさえ、ゴール指向エージェントにとっては脆弱で恣意的な制約として機能してしまうと主張する。

ユーダイモニック合理性の特徴は「xをx的に促進する（promote x x-ingly）」という公式に集約される。例えば、数学者は数学的卓越性を数学的卓越性をもって促進し、友人は友情を友情的に育む。この構造では手段と目的の二分法が自然には現れず、行動はメロディの音符や計算のタイムステップのように、より大きなプラクティスの要素として意味を持つ。

Terry Taoの数学研究の記述を事例として、ユーダイモニック合理性の具体的作動を分析する。数学者が「優れた数学をする」ために何をするかを問うと、それは明示的な目標関数の最適化ではなく、数学的プラクティス内の行動評価基準に従った有機的な探索であることが示される。

AIアライメントへの含意として、著者はユーダイモニック合理性がEA型最適化より「自然（natural）」な形式である可能性を論じる。ここでの「自然」とは規範的意味ではなく、安定性・学習容易性・アルゴリズム複雑度の低さ・異なる合理的動物種間での収束的文化進化の可能性・ニューラルネットワークとの収束進化可能性を指す。もしユーダイモニック合理性がこの意味で自然であれば、MLの訓練プロセスによって比較的安全かつ容易にターゲット化できるアライメント手法の候補となる。

監査エージェント開発への示唆として、corrigibility（修正可能性）をゴールとして設計するのではなく、監査プラクティス内の行動傾向として埋め込む設計思想が参照可能である。LangGraphやReActベースのエージェントにおいて、各ステップの評価基準を「監査的卓越性を監査的に促進する」プラクティスとして定式化することで、ルールベースの制約より堅牢な安全性が得られる可能性がある。

## アイデア

- 「promote x x-ingly」公式：安全特性（harmlessness, corrigibility）をゴールとして定義するのではなく、エージェントのプラクティス内の動的傾向として実装する設計原則として応用可能
- 型の不一致（type mismatch）問題：EA型最適化エージェントに人間の価値を効用関数として渡すこと自体が構造的に失敗しやすいという診断は、現行のRLHF/DPOベースのアライメント手法の根本的限界を示唆する
- 「自然さ（naturalness）」の技術的概念化：安定性・学習容易性・アルゴリズム複雑度をもってエージェントアーキテクチャの良さを評価する基準として、ベンチマーク性能とは独立した設計指標となりうる
