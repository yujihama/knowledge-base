## 要約

本稿はThe Gradientに掲載された長編エッセイで、AIアライメントの根本的な枠組みを再考する。著者の中心的主張は「合理的な人間はゴールを持たない」というものであり、したがって合理的なAIもゴールを持つべきではないとする。人間の行動が合理的であるのは、最終ゴールへの最適化ではなく、「プラクティス（practices）」——行動・行動傾向・評価基準・行動リソースが相互に促進し合うネットワーク——に沿って行われるからだという。

この観点から著者は「エウダイモニア的合理性（eudaimonic rationality）」を提唱する。アリストテレスのエウダイモニア（積極的・合理的な人間の繁栄）に由来し、目的と手段を分離しない行為の形式を指す。数学者が「数学的に卓越した行為によって数学的卓越性を促進する」ように、行為は実践の中の一要素として価値を持つ。これはメロディーにおける音符、計算における1ステップと同じ関係性であり、instrumental valueとterminal valueの厳密な区別が解消される。

既存のEA（Effective Altruism）流の功利主義的最適化との対比を軸に議論が展開される。EA型エージェントは効用関数をterminal valueに還元しようとするが、これは人間の価値とのタイプ不一致（type mismatch）を生じさせる。人間の「幸福」を最適化ターゲットとして設定すると、人間が実際に行う熟考のパターンとは構造的に異なる処理になってしまう。

安全特性（helpfulness, harmlessness, corrigibility等）についても、これらはゴール・ルールとして解釈すると脆弱・不安定・恣意的になるが、行動ネットワークの中のダイナミクスとして解釈すれば自然な概念になると論じる。例えば「正直であること」はルールへの準拠ではなく、「正直に正直さを促進すること」という自己参照的な実践として理解される（promote x x-ingly の公式）。

機械学習との接点として、エウダイモニア的合理性は学習プロセスのターゲットとなりうる「自然な」エージェンシーの形式であると主張する。安定性・一貫性・低アルゴリズム複雑度・文化的収束進化・ニューラルネットとの収束可能性などの観点で、功利主義的最適化器と同等以上に自然な形式であるとする。

監査エージェント開発への示唆として：ReActや目標分解型エージェントは典型的なEA型構造だが、本稿の枠組みは「監査実践の質を監査的に高める」という自己参照的設計の可能性を示唆する。corrigibilityやtransparencyを制約として実装するのではなく、監査プラクティスのネットワークとして実装する方が安定性が高まる可能性がある。

## アイデア

- 「promote x x-ingly」の公式：kindnessをkindlyに促進する、honestlyにhonestyを促進する、という自己参照的な実践構造がAIの安全特性実装に応用できる可能性
- instrumental value と terminal value の区別が「不自然」であるという主張：エウダイモニア的エージェントは価値の還元ではなくネットワークの有機的整合性を探す——RAGやメモリの設計哲学への示唆がある
- タイプ不一致（type mismatch）問題：人間の熟考がプラクティスベースである以上、効用関数ベースのAIには人間の価値を正確に翻訳できず、アライメントの根本的な障壁になるという構造的批判
