## 要約

本記事は金融工学の基幹理論であるブラック-ショールズ（BS）方程式を数式レベルで導出し、Pythonで完全実装する技術解説。前提として幾何ブラウン運動（GBM）と伊藤の補題を活用する。

導出の核心は「複製ポートフォリオ」アプローチにある。オプション価値V(t,S)に伊藤の補題を適用し、Π = V - (∂V/∂S)・S というデルタヘッジポートフォリオを構成すると、確率項dWが消去されリスクフリーなポートフォリオが得られる。無裁定条件によりその収益率はリスクフリーレートrに等しくなければならず、結果として∂V/∂t + rS・∂V/∂S + (1/2)σ²S²・∂²V/∂S² = rV というBS偏微分方程式が導かれる。注目点は原資産のドリフトμが消え、ボラティリティσとリスクフリーレートrのみでオプション価格が決定されること（リスク中立評価の本質）。

境界条件V(T,S)=(S-K)⁺のもとで解析解を求めると、コールオプション価格C = SΦ(d₁) - Ke^{-r(T-t)}Φ(d₂)、プットP = Ke^{-r(T-t)}Φ(-d₂) - SΦ(-d₁)が得られる。d₁ = [ln(S/K)+(r+σ²/2)(T-t)]/(σ√(T-t))、d₂ = d₁ - σ√(T-t)。

グリークスとしてDelta(∂V/∂S)、Gamma(∂²V/∂S²)、Theta(∂V/∂t)、Vega(∂V/∂σ)、Rho(∂V/∂r)を全て解析的に導出・実装。ガンマはATM付近で最大となるなど各指標の直感的解釈も提供される。

最後にモンテカルロ法（GBMによる株価パスのシミュレーション）でオプション価格を数値的に計算し、BS解析解と比較することで理論と数値計算の一致を確認。numpy・scipy・matplotlibのみで完結する実装で、パラメータS=100、K=100、r=0.05、σ=0.2を標準ケースとして価格曲面のヒートマップも可視化している。

監査・リスク管理視点への応用可能性は低いが、金融リスクモデルの定量評価・バリデーション手法として、モデルリスク管理（MRM）や内部監査でデリバティブ評価モデルをレビューする際の数理的基盤となる。

## アイデア

- 複製ポートフォリオによるdW消去がリスク中立評価の核心であり、ドリフトμが式から消えることで投資家のリスク選好に依存しない普遍的な価格付けが実現される点
- グリークス（Delta, Gamma, Vega等）はオプション価格の偏微分として解析的に導出でき、ヘッジ戦略の数理的基盤になる点—特にGammaがATM付近で最大となる非線形性はヘッジコスト管理に直結する
- BS解析解とモンテカルロ数値解の比較検証パターンは、LLM評価でのreference-based vs reference-freeの比較構造と類似しており、数値モデルのバリデーション手法として汎用的に応用できる点
